Admin Tool

Von der PHP-Anwendung zum Web Service

Diese Anleitung erläutert die Verwendung des Admin-Tools zur Erstellung von Web Services aus bestehenden PHP-Anwendungen. Diese Anwendungen müssen lediglich gemäß dem ebenfalls im Anhang befindlichen Style-Guide dokumeniert und als Web Services gekennzeichnet werden.

Die PHP-Datei queryLink, ein Teil der Notfalllösung (Andi: vlt besser: Beispielimplementierung) zum tele-TASK-Framework und dessen Methoden werden hier als Beispiel verwendet.

<?php //***** queryLink ******************************************************** /** * queryLink - provides basic queries on table links of the Teletask DB * @webservice */ class queryLink extends DbCollection { //========================================================================= /** * Returns an array of strings, index represents link id, value the name * * @webmethod * @return String[] */ public function getIdAndNameOfAllLinks() { $sql = 'SELECT LINK.name, LINK.id FROM links LINK'; $this->_lastResult = $this->_db->Execute($sql); if (isset($this->_lastResult)) { while (!$this->_lastResult->EOF) { $item = new Link(-1, $this->_lastResult->fields); $result[$item->getID()] = $item->getName(); $this->_lastResult->MoveNext(); } } return $result; } }

Klassen und Methoden, die als Web Service verfügbar gemacht werden sollen,
müssen mit @webservice beziehungsweise @webmethod gekennzeichnet werden.

Wählen sie die Methoden aus, die mittels WebService veröffentlicht werden sollen.


Methoden:







Nach Auswahl der zu veröffentlichen Klasse, erhalten Sie eine Übersicht über alle in der Klasse vorkommenden und mit dem @webmethod Tag versehenden Methoden.
Sie können wiederum jene Methoden auswählen, die veröffentlich werden sollen.

Wählen Sie 'Speichern' werden Informationen über die Klasse und ihre Methoden in einer Datenbank abgelegt. Dabei wird vermerkt, ob die Methode veröffentlich werden soll. Sie können für jede Methode einen individuellen Kommentar in der Datenbank speichern, der später in die WSDL-Datei eingebracht wird. Der Source-Kommentar wird ebenfalls gespeichert.
